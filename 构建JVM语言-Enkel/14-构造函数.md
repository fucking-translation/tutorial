## 语法

Enkel的构造函数的声明与Java的的调用语法类似（除了默认构造函数和命名参数的特性之外）。

声明示例：

```groovy
Cat ( String name ) {
}
```

调用：

```groovy
new Cat ( "Molly" )
```

## 语法变更

在Java中，构造函数的声明语法仅仅是一个没有返回值类型的函数声明。事实证明，Enkel方法声明不需要指定返回值（如果方法返回空）。所以这里不需要为了构造函数声明来指定一条新的规则。
